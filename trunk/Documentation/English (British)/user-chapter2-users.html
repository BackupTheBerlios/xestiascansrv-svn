2.2 User Management

To ensure users do not use or use only certain scanners, output format modules and export format modules, user management is required.

User management can be done (as a user with administrative privileges) by clicking on the User Management link on the top menu.

2.1.1 Add a User

To add a user to the user management list, click on the Add Users button on the user list. This will then display the following settings to enter for the user under the User Details section:

Setting Description
User name Specifies the username of the account.
Name Specifies the name of the person associated with this account.
Administrator Privileges Specifies if the account has administrator privileges. An account with administrator privileges is able to manage the users who can access scanners and can the settings for the scanner server software itself.
Account Enabled Specifies if the account should be enabled or disabled. If the account enabled button is not checked then this account will not be able to login.
Password Specifies the password to use.
Confirm Password Confirms the password to use by entering it a second time.

The next section is the Scanner List which lists the permissions for the scanners. To allow access to a scanner, simply check the Allow Access checkbox next to the name of the scanner.

The third section is the Output Format Module List which lists the permissions for the output format modules. To allow access to an output format module, simply check the Allow Access checkbox next to the name of the scanner.

The final section is the Export Format Module List which lists the permissions for the export format modules. To allow access to an export format module, simply check the Allow Access checkbox next to the name of the scanner.

To confirm the settings given, click on the Add User button at the bottom of the screen. To clear the values and start again click on the Clear values also at the bottom of the screen.

2.1.2 Edit a User

To edit a user’s settings, select a user on the user management list and click on the Edit link in the List Users screen. This will display the following settings to enter for the user under the User Details section:
Setting Description
User name Specifies the username of the account.
Name Specifies the name of the person associated with this account.
Administrator Privileges Specifies if the account has administrator privileges. An account with administrator privileges is able to manage the users who can access scanners and can the settings for the scanner server software itself.
Account Enabled Specifies if the account should be enabled or disabled. If the account enabled button is not checked then this account will not be able to login.
New Password Specifies the new password to use.
Confirm New Password Confirms the new password to use by entering it a second time.

The next section is the Scanner List which lists the permissions for the scanners. It will also display the permissions for any scanners which aren’t plugged in but the user can access when they are plugged in again. To allow access to a scanner, simply check the Allow Access checkbox next to the name of the scanner.

The third section is the Output Format Module List which lists the permissions for the output format modules. To allow access to an output format module, simply check the Allow Access checkbox next to the name of the scanner.

The final section is the Export Format Module List which lists the permissions for the export format modules. To allow access to an export format module, simply check the Allow Access checkbox next to the name of the scanner.

To save the changed settings for the user, click on the Edit User button. To revert the settings back to their original settings click on the Clear values button.

Note

If the username is changed, the user under the previous username will be logged out and will have to login using the new username.


2.1.3 Delete a User

To delete a user, click on the Delete link next to the user you want to delete on the user management list and to confirm deletion click on the ‘Yes, delete user’ button. This will remove the user from the list.

Deleting the administrative account used to manage users is not recommended as the only way to recover from this is to rerun the Multiuser Installer script again.

2.1.4 Logout All Users

If you need to logout all users for the purposes of maintenance or in the event of an emergency, you are able to logout all users. To do this, click on the Logout All Users button on the users list and then click on ‘Logout All Users’ to confirm this action. This will log everyone out (including the user requesting this action). You will need to log back in again to continue.
