1.11 xestiascan_processfilename

xestiascan_processfilename processes a name and turns it into a filename that can be used by Xestia Scanner Server.

Parameters:

xestiascan_processfilename(text);


Usage:

$filename = xestiascan_processfilename(“For the 3rd time already”);


text
Specifies the text to create a filename with.
