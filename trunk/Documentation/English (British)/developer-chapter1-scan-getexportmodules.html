1.17 xestiascan_scan_getexportmodules

xestiascan_scan_getexportmodules gets the list of available export modules in the Modules/Export directory and returns the list as an array.

Parameters:

xestiascan_scan_getoutputmodules();


Usage:

@outputmodules = xestiascan_scan_getoutputmodules();
