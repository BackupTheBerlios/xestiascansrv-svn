1.14 xestiascan_auth_authenticate

xestiascan_auth_authenticate prints out a form for authenticating and logging into the scanner server.

Parameters:

xestiascan_auth_authenticate(authfailure);


Usage:

$pagedata = xestiascan_auth_authenticate();


authfailure
A failure occurred whilst authenticating. A value of 1 would print an error message.
