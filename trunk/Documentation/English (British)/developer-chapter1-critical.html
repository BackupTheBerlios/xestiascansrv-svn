1.5 xestiascan_critical

xestiascan_critical processes critical errors when using Xestia Scanner Server.

Parameters:

xestiascan_critical(errortype);


Usage:

xestiascan_critical(“generic”);


errortype
Specifies the critical error message to use.

For more information on the error messages used look at the xestiascan_critical subroutine in the Xestia Scanner Server source code.
