1.31 xestiascan_settings_getlanguages

xestiascan_settings_getlanguages gets a list of language files from the lang directory and returns them as an array.

Parameters:

xestiascan_settings_getpresmodules();


Usage:

@languages = xestiascan_settings_getlanguages();
