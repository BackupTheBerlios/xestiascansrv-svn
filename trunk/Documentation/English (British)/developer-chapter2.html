Chapter 2: Presentation Module

The subroutine listed here in this chapter are used by the presentation module. The presentation module outputs the page data in Xestia Scanner Server as the specified format. An example of this is the HTML4S module which writes the page in the HTML 4.0 Strict Format.

When the presentation module is loaded the subroutines for it can be called from the $xestiascan_presmodule scalar.
