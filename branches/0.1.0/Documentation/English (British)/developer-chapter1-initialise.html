1.1 xestiascan_initialise

xestiascan_initialise loads some variables before they are used throughout Xestia Scanner Server (such as the script filename). This subroutines does not need to be called anywhere else.

Parameters:

xestiascan_initialise();


Usage:

xestiascan_initialise();
