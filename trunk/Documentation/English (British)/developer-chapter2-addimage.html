2.32 addimage

The addimage subroutine adds an image.

Parameters:

$xestiascan_presmodule->addimage(image, options);


Usage:

$xestiascan_presmodule->addimage("image.png", { Width => "100", Height => "100", Description => "This is an image." });


image
Specifies the filename of the image to use.

options
Specifies the following options as a hash in any order.

Setting Description
Style Specifies the CSS style to use for the image.
Description Specifies the description of the image.
Width Specifies the width of the image.
Height Specifies the height of the image.
