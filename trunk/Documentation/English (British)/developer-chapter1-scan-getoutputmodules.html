1.16 xestiascan_scan_getoutputmodules

xestiascan_scan_getoutputmodules gets the list of available output modules in the Modules/Output directory and returns the list as an array.

Parameters:

xestiascan_scan_getoutputmodules();


Usage:

@outputmodules = xestiascan_scan_getoutputmodules();
