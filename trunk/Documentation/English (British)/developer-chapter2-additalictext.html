2.35 additalictext

Parameters:

$xestiascan_presmodule->additalictext(text, options);


Usage:

$xestiascan_presmodule->additalictext("This is italic formatted text.", { Style => "italictext" });


text
Specifies the italic text to use.

options
Specifies the following options as a hash in any order.

Setting Description
Style Specifies the CSS style to use.
