1.21 xestiascan_scan_getscannerlist

xestiascan_scan_getscannerlist gets a list of available scanners and returns them as a hash.

Parameters:

xestiascan_scan_getscannerlist();


Usage:

%scannerlist = xestiascan_scan_getscannerlist();


Each scanner has three values:

Setting Value
name The SANE ID of the scanner.
model The model of the scanner.
vendor The vendor of the scanner.
