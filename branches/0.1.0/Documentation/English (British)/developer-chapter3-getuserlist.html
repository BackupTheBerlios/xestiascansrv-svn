3.10 getuserlist

Gets the user list. Returns as a hash.

Parameters:

$xestiascan_authmodule->getuserlist();


Usage:

%userlist = $xestiascan_authmodule->getuserlist(options);


options
Specifies the following options as a hash.

Setting Description
Reduced Gets a reduced version of the user list.
ShowDeactivated Show users that are deactivated from the list.


The hash returns the following data for each user.

Setting Description
username Specifies the username of the user.
name Specifies the name of the user.
deactivated Specifies if the user account is enabled or disabled.
