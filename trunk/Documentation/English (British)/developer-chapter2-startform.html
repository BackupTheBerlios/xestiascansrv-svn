2.18 startform

The startform subroutine starts a form.

Parameters:

$xestiascan_presmodule->startform(action, method);


Usage:

$xestiascan_presmodule->startform("xsdss.cgi", "POST");


action
Specifies where the form data should be sent to.

method
Specifies the method of posting the data.

Setting Description
POST Posts the form data. The data does not appear in the address bar.
GET Gets the form data. The data appears in the address bar.
