1.29 xestiascan_settings_getauthmodules

xestiascan_settings_getauthmodules gets a list of authentication modules from the Modules/Auth directory and returns them as an array.

Parameters:

xestiascan_settings_getauthmodules();


Usage:

@authmodules = xestiascan_settings_getauthmodules();
