Welcome
This is the documentation for Xestia Scanner Server and it is split into three parts:
  • User Documentation
    The User Documentation provides information on how to install and use your Xestia Scanner Server installation.
  • Tutorial Documentation
    The Tutorial Documentation provides some tutorials on how to use certain featurs of Xestia Scanner Server.
  • Developer Documentation
    The Developer Documentation provides information on how to create additional presentation modules, authentication modules, export modules, output modules and information on translating Xestia Scanner Server into another language.
The Xestia Scanner Server Documentation Guide provides information about the symbols that appear while reading the documentation.
