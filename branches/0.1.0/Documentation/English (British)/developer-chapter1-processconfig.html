1.12 xestiascan_processconfig

xestiascan_processconfig processes a configuration file (such the Xestia Scanner Server configuration file or a Xestia Scanner Server).

Parameters:

kiriwrite_processconfig(data);


Usage:

%config = kiriwrite_processconfig(@config);


data
Processes a configuration file that is formatted as an array and returns the configuration as a hash. This is much easier to use as specific configuration data does not need to be on a specific line.
