2.24 addinputbox

The addinputbox subroutine adds an input box.

Parameters:

$xestiascan_presmodule->addinputbox(inputboxname, options);


Usage:

$xestiascan_presmodule->addinputbox("inputbox", { Size => "64", MaxLength => "128", Style => "inputbox", Value => "Default", Password => 0 });


inputboxname
Specifies the input box name to use.

options
Specifies the following options as a hash in any order.

Setting Description
Size Specifies the size of the input box.
MaxLength Specifies the maximum length of the input box.
Style Specifies the CSS style of the input box.
Value Specifies the value of the input box.
Password Specifies if this input box is for a password.
ReadOnly Specifies if the value should be read only.
