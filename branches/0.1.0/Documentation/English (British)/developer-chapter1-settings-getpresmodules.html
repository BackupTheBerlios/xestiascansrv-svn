1.30 xestiascan_settings_getpresmodules

xestiascan_settings_getpresmodules gets a list of presentation modules from the Modules/Presentation directory and returns them as an array.

Parameters:

xestiascan_settings_getpresmodules();


Usage:

@presmodules = xestiascan_settings_getpresmodules();
