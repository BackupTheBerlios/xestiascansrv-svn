Chapter 3: Authentication Module

The subroutines listed here in this chapter are used by the authentication module. The authentication module allows users to be authenticated to use the scanner server.

An example of a server-based authentication module is the PostgreSQL module (which is a database module for the PostgreSQL database servers).

When the authentication module is loaded the subroutines for it can be called from the $xestiascan_authmodule scalar.
