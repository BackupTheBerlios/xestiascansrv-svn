Chapter 1: Internal Xestia Scanner Server Subroutines

The subroutines listed here are used internally by Xestia Scanner Server. Typically most of the subroutines here shouldn't be used anywhere else other than the xestiascan_filepermissions and xestiascan_fileexists subroutines which can be used for checking if files exist and have valid permissions set.
