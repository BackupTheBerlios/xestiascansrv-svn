1.32 xestiascan_settings_view

xestiascan_settings_view displays the current settings of the Xestia Scanner Server installation. The current settings will only be displayed if the logged in user has administrative privileges.

Parameters:

xestiascan_settings_view();


Usage:

$pagedata = xestiascan_settings_view();
