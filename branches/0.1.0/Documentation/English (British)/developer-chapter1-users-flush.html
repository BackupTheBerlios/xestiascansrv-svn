1.28 xestiascan_users_flush

xestiascan_users_flush logs everyone out of the Xestia Scanner Server installation.

Parameters:

xestiascan_users_flush(confirm);


Usage:

$pagedata = xestiascan_users_flush(confirm);


confirm
Confirms the action to log everyone out.
