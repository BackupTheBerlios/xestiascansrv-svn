2.39 additem

Parameters:

$xestiascan_presmodule->additem(text, options);


Usage:

$xestiascan_presmodule->additem("This is an item", { Style => "itemstyle" });


text
Specifies the text to use for the item.

options
Specifies the following options as a hash in any order.

Setting Description
Style Specifies the CSS style to use.
