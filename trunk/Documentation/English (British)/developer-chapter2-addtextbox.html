2.25 addtextbox

The addtextbox subroutine adds multiple line text box.

Parameters:

$xestiascan_presmodule->addtextbox(textboxname, options);


Usage:

$xestiascan_presmodule->addtextbox("textbox", { Columns => 25, Rows => 5, Style => "textbox", Value => "This is a text box." });


textboxname
Specifies the name of the multiple line text box.

options
Specifies the following options as a hash in any order.

Setting Description
Columns Specifies the width of the multiple line text box.
Rows Specifies the height of the multiple line text box.
Style Specifies the CSS style of the multiple line text box.
Value Specifies the value of the multiple line text box.
ReadOnly Specifies if the text box is read only.
